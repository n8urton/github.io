Install Spacemacs & GnuPG on Windows 10

2019-09-08
#pgp  #windows  #org  #encryption 

Requirements

  • chocolatey
  • git

Using the chocolatey package manager and git, we are going to get everything up and running.

choco install

  1. First, we need to open powershell or cmd in administrative mode

  2. Next, install Emacs, the Source Code Pro font, and gpg4win with chocolatey with the following command

choco install spacemacs gpg4win sourcecodepro

Now to add Spacemacs to our Start Menu

  1. Press win key and search for runemacs.exe and open file location or use File Explorer to navigate to C:\ProgramData\chocolatey\bin
  2. Using the Run commad (Windows key + R), type %AppData%\Microsoft\Windows\Start Menu\Programs and click OK to open a second File Explorer window
  3. Right-click, hold, drag and drop the runemacs.exe file from the \chocolatey\bin window to the \Programs window
  4. Select Create shortcuts here from the context menu
  5. Right-click the runemacs.exe shortcut, select Rename to rename the shortcut to Spacemacs
  6. Right-click the Spacemacs shortcut and select Properties. On the Shortcut tab, find the Start in setting and change to C:\Users\$UNAME or another directory you want Spacemacs to start in. Click Apply
  7. Check to see if Spacemacs is now in your start menu

GnuPG setup

Open Kleopatra from the start menu. Use the File tab to create a New Key Pair or import your existing public/private keys

Add GnuPG to Spacemacs

Add the following to defun dotspacemacs/user-config in your .spacemacs config file

(custom-set-variables
 '(epg-gpg-home-directory "c:/Users/MYUSER/AppData/Roaming/gnupg")
 '(epg-gpg-program "C:/Progra~2/gnupg/bin/gpg.exe")
 '(epg-gpgconf-program "c:/progra~2/gnupg/bin/gpgconf.exe")
)

Create a .gpg file and open with Spacemacs

  1. In Spacemacs, enter Space f f to open a new file with the name test.gpg
  2. Write some stuff
  3. In Visual Mode, enter :w to save
  4. Follow the prompts to select the key you generated in Kleopatra, pressing Alt and typing epa-mark-key to mark or Alt and typing epa-unmark-key to deselect. If no keys are selected, you will be prompted for a password to symetrically encrypt the file

Sync

Just save the file in your favorite cloud. Be sure to securley migrate your keys to any other machines you’ll want to access the file from.

Fun Fact

Anyone can send you an encrypted file using your public key that will require your unique private key to decrypt.
